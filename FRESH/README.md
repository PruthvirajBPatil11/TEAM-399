# FRESH Folder - HTML Files

## Overview

This folder contains HTML files for the emergency ambulance management system interface.

## Files

### 1. index.html
Landing page for the emergency dispatch system.

### 2. Ambulance1.html
Ambulance driver portal with live notification system.
- Polls Baserow API for new assignments
- Displays emergency details in modal popup
- Driver acknowledgment system

### 3. Ambulance2.html
Alternative ambulance driver interface.

### 4. controlroom.html
Control room map viewer for monitoring emergency locations.
- Real-time map with emergency markers
- Baserow API integration
- Location tracking and management

## Usage

Open any HTML file in a web browser to use the interface.

**Note:** Requires Baserow API credentials to be configured in the HTML files.
